# 14.2 Kvantdatorer och framtidens beräkningskraft

Samtidigt som vi ser en enorm utveckling inom traditionell halvledarteknik (se kapitel 1.2), närmar vi oss den fysiska gränsen för hur små vi kan göra transistorer på ett kiselchip. När komponenterna blir så små att de bara består av enstaka atomer börjar kvantfysikaliska effekter störa de elektriska signalerna. Men istället för att se detta enbart som ett hinder, har forskare skapat en helt ny typ av dator: kvantdatorn. För en it-tekniker i framtiden representerar detta inte bara snabbare hårdvara, utan ett helt nytt sätt att hantera information som kan komma att runda många av de säkerhetsprinciper vi förlitar oss på idag.

En traditionell dator arbetar med bitar som antingen är 1 eller 0. En kvantdator använder istället kvantbitar, eller qubits. Genom ett fenomen som kallas superposition kan en qubit vara 1, 0 eller båda tillstånden samtidigt. Dessutom kan qubits sammanflätas genom "entanglement", vilket innebär att tillståndet hos en qubit direkt påverkar en annan, oavsett avstånd. Detta gör att en kvantdator kan utföra massiva beräkningar parallellt på ett sätt som en superdator aldrig skulle kunna drömma om. Problem som skulle ta miljarder år för dagens mest kraftfulla datorer att lösa kan en kvantdator teoretiskt hantera på några minuter.

Inom it-säkerhet är kvantdatorn både ett hopp och ett hot. Det största hotet ligger i att kvantdatorer är extremt effektiva på att lösa de matematiska problem som ligger till grund för dagens asymmetriska kryptering, såsom RSA (se kapitel 11.3). Den dag en tillräckligt kraftfull kvantdator existerar kan i princip all nuvarande krypterad kommunikation som sparats av angripare låsas upp. Detta har lett till utvecklingen av "Post-Quantum Cryptography" (PQC), vilket är nya krypteringsalgoritmer som är designade för att vara resistenta även mot kvantattacker (https://www.nist.gov/post-quantum-cryptography).

Trots sin enorma potential är kvantdatorer idag extremt känsliga. De kräver temperaturer nära den absoluta nollpunkten för att fungera, vilket innebär att de inte kommer att ersätta din laptop eller din smartphone inom överskådlig tid. Snarare kommer vi att se dem som specialiserade resurser i molnet, där de används för specifika uppgifter som att simulera nya material, optimera komplexa logistikflöden eller knäcka avancerade koder. Som tekniker i framtiden kommer din kontakt med dessa system troligen ske via API:er och molntjänster, snarare än genom fysiskt underhåll av själva kvantprocessorn.
