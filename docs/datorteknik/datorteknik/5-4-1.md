# 5.4.1 – Förberedelser

✅ Kontrollera systemkrav

Processor: 64-bitars, 1 GHz eller snabbare, minst 2 kärnor.

RAM: Minst 4 GB (rekommenderat 8+ GB).

Lagring: 64 GB eller mer.

TPM 2.0 och Secure Boot (krav för Windows 11).
