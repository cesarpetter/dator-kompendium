# 2.7.3 – Nackdelar

Kan hindra installation av osignerade operativsystem

Kräver certifikatunderhåll (tillverkare kan dra tillbaka certifikat)

Kan behöva inaktiveras för vissa Linux-distributioner eller specialsystem
