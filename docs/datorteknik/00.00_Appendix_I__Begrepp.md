# Appendix I – Begrepp

3-2-1-regel: Tre kopior, två olika medier, en kopia off-site – grundregel för backup.

A/B-test: Metod för att jämföra två varianter och mäta effekt.

AC-ström: Växelström; vägguttagets ström som PSU omvandlar till DC.

ACL (Access Control List): Lista som styr åtkomst till fil, mapp eller resurs.

ADB (Android Debug Bridge): Verktyg för att styra/felsöka Android-enheter.

ADK (Assessment and Deployment Kit): Microsoft-verktyg för Windows-deployment.

AHCI: SATA-drivrutinsläge som möjliggör NCQ och hot swap.

AI (Artificiell Intelligens): Teknik som syftar till att skapa maskiner med mänsklig kognitiv förmåga och problemlösningsförmåga.

AIO-kylare: “All-in-one” vattenkylning med pump/radiator i ett kit.

AM4: AMD:s äldre processorsockel för Ryzen-processorer som använder DDR4-minne.

AM5: AMD:s moderna processorsockel för Ryzen-processorer som använder DDR5-minne.

Anisotropisk filtrering: Teknik som skärper texturer när de betraktas i sneda vinklar.

Anti-aliasing (AA): Kantutjämning som minskar “trappstegseffekter” i digital grafik.

APFS: Apples filsystem optimerat för SSD med stöd för snapshots och kryptering.

AppArmor: Linux-säkerhetsmodul som begränsar enskilda programs behörigheter och resurstillgång.

ARC (Automatic Reference Counting): Minneshantering som sker genom att räkna antalet referenser till ett objekt.

ARP: Protokoll som översätter en IP-adress till en fysisk MAC-adress i ett lokalt nätverk.

ARP-cache: En tabell i datorn eller routern över nyligen upplösta IP-till-MAC-par.

ATX: Den vanligaste formfaktorn för moderkort och chassin för stationära datorer.

Auto-negotiation: Automatisk förhandling av nätverkslänkens hastighet och duplexläge.

AVX/AVX2/AVX-512: Vektor-instruktioner i processorn som accelererar tunga parallella beräkningar.

Backup: En extra kopia av data som sparas för att möjliggöra återställning vid förlust eller skada.

Bandbredd: Den maximala mängden data som kan överföras per sekund över en nätverksförbindelse.

BCD (Boot Configuration Data): Microsofts databas för startkonfiguration i moderna Windows-versioner.

BGP (Border Gateway Protocol): Det primära routingprotokollet som används för att styra trafik mellan olika autonoma system på internet.

Bifröst: En specifik GPU-arkitektur från ARM designad för mobila enheter.

BIOS: Äldre typ av firmware som initierar hårdvaran och startar operativsystemet.

BIOS-setup: Den konfigurationsmeny där man ändrar inställningar för BIOS eller UEFI.

BitLocker: Programvara för fullständig diskkryptering inbyggd i Windows.

Block: Den minsta enheten för dataöverföring (I/O) i lagringssammanhang.

Bluetooth: Trådlös teknik för kommunikation över korta avstånd mellan tillbehör och datorer.

Bottleneck: En komponent som är betydligt långsammare än övriga och därmed begränsar systemets totala prestanda.

Brandvägg (Firewall): Mjukvara eller hårdvara som kontrollerar inkommande och utgående nätverkstrafik baserat på förinställda säkerhetsregler.

Btrfs: Ett avancerat Linux-filsystem med stöd för Copy-on-Write, snapshots och checksummor.

Cache: Ett mycket snabbt minne placerat nära CPU eller disk för att lagra ofta använda data.

CAS-latens: Fördröjningen i antal klockcykler innan arbetsminnet svarar på en förfrågan.

CDN (Content Delivery Network): Ett globalt nätverk av servrar som distribuerar innehåll geografiskt nära användaren för högre hastighet.

Checksumma: Ett matematiskt värde som används för att kontrollera att data inte har förändrats eller skadats.

Chiplet: En modulär designmetod där en processor består av flera mindre kretsar som samarbetar.

Chipset: De styrkretsar på moderkortet som sköter kommunikationen mellan CPU och övriga enheter.

CI/CD: Kontinuerlig integrering och leverans av programvara för att snabba upp utvecklingsprocessen.

CIFS/SMB: Protokoll för att dela ut mappar och enheter i Windows-baserade nätverk.

CMOS-batteri: Ett litet batteri på moderkortet som ger ström åt klockan och UEFI-inställningarna när datorn är avstängd.

CMR (Conventional Magnetic Recording): Traditionell teknik för hårddiskinspelning där varje dataspår ligger separat.

Coil whine: Ett högfrekvent surrande ljud som kan uppstå från elektriska spolar när de belastas hårt.

Container: En lättviktsform av virtualisering där applikationer körs i isolerade miljöer som delar samma operativsystemkärna.

Copy-on-Write (CoW): En teknik som skapar en ny kopia av data vid ändring istället för att skriva över originalet.

CPU: Central Processing Unit; processorn som utför alla instruktioner i datorn.

CRC (Cyclic Redundancy Check): En metod för felkontroll som effektivt upptäcker bitfel i digital data.

CVE: En katalog över kända sårbarheter i mjuk- och hårdvara, där varje sårbarhet har ett unikt ID-nummer.

CVSS: Ett standardiserat poängsystem som används för att gradera hur allvarlig en säkerhetssårbarhet är.

Daisy chaining: Metod för att kedjekoppla flera enheter efter varandra i en serie via en enda port.

DAS (Direct Attached Storage): Lagringsenhet som är direkt ansluten till en enskild dator via exempelvis USB eller eSATA.

DEDUP (Deduplicering): En teknik som sparar utrymme genom att identifiera och ta bort identiska datakopior.

Default Gateway: Adressen till den enhet som skickar trafik vidare till andra nätverk när mottagaren inte finns i det lokala nätverket.

Defragmentering: Processen att samla ihop filfragment på en mekanisk hårddisk för att snabba upp läshastigheten.

DHCP: Protokoll som automatiskt tilldelar IP-adresser och nätverksinställningar till enheter.

DHCP-snooping: En säkerhetsfunktion i en nätverksswitch som förhindrar att falska DHCP-servrar delar ut adresser.

Digital forensik: Vetenskapen att säkra, analysera och presentera digitala bevis från datorer, mobiler och nätverk.

DIMM: Den fysiska typen av RAM-modul som används i stationära datorer.

DISKPART: Ett kommandobaserat verktyg i Windows för att hantera diskar, partitioner och volymer.

DisplayPort: Ett modernt digitalt videogränssnitt med hög bandbredd och stöd för daisy chain.

DisplayPort Alt Mode: Teknik för att skicka videosignaler över en USB-C-kabel.

DISM: Windows-verktyg för att reparera systemfiler och hantera Windows-images.

DLSS: Nvidias AI-baserade teknik som skalar upp lägre upplösningar för att höja FPS med bibehållen bildkvalitet.

DNS: Systemet som översätter mänskligt läsbara domännamn (t.ex. google.se) till IP-adresser.

Dockningsstation: En enhet som ansluts med en kabel och ger tillgång till extra portar, video och laddning.

DRAM-cache (SSD): Ett inbyggt snabbminne i en SSD som används för att snabba upp hantering av metadata.

DRAM-lös SSD (HMB): En billigare typ av SSD som lånar en del av datorns RAM-minne för att hantera sina tabeller.

DREAD: En modell som används vid hotanalys för att bedöma och prioritera olika säkerhetsrisker.

Dual channel: Teknik som använder två minneskanaler samtidigt för att dubblera bandbredden till arbetsminnet.

DVI: Ett äldre digitalt videogränssnitt som numera oftast ersatts av HDMI eller DisplayPort.

DWM (Desktop Window Manager): Den komponent i Windows som hanterar renderingen av skrivbordet och fönstereffekter.

DWPD: Drive Writes Per Day; ett mått på hur mycket data som kan skrivas till en SSD varje dag under dess garantitid.

ECC (felkorrigering): Specialiserat minne eller lagring som kan upptäcka och automatiskt korrigera bitfel.

Edge Computing: En beräkningsmodell där data bearbetas nära källan (vid nätverkets "kant") istället för i ett centralt moln.

EFI-systempartition (ESP): En specifik partition på hårddisken som innehåller startfilerna för UEFI.

eGPU: Ett externt grafikkort som ansluts till en dator via exempelvis Thunderbolt eller USB4.

Egress-filtrering: Brandväggsregler som kontrollerar och begränsar trafik som lämnar det interna nätverket.

Eller-villkor (OR): Ett logiskt villkor som är sant om minst ett av flera ingående kriterier uppfylls.

E-postprotokoll: Samlingsnamn för SMTP (skicka), IMAP (synkronisera) och POP3 (hämta).

EPP/XMP/EXPO: Fördefinierade prestandaprofiler i RAM-minnet som förenklar överklockning och inställningar.

Event Viewer (Händelselogg): Ett inbyggt verktyg i Windows för att granska loggar från systemet och applikationer.

exFAT: Ett filsystem designat för externa medier som minneskort och USB-minnen med stöd för stora filer.

Fabriksåterställning: Processen att återställa ett operativsystem till dess ursprungliga tillstånd.

Failover: En funktion som automatiskt växlar över till en reservresurs om den primära resursen slutar fungera.

FAT32: Ett äldre filsystem med mycket hög kompatibilitet men begränsningar i filstorlek (max 4 GB).

Fibre Channel: Ett höghastighetsprotokoll som främst används för att koppla samman servrar med lagringsnät (SAN).

Firmware-uppdatering: Installation av ny mjukvara i en hårdvarukomponent för att rätta fel eller lägga till funktioner.

Fjärrskrivbord (RDP): Microsofts protokoll för att fjärrstyra en annan Windows-dator grafiskt.

FPS: Bilder per sekund; ett mått på hur mjukt en animation eller ett spel flyter på skärmen.

Frame pacing: Teknik för att säkerställa att tiden mellan varje bildruta är så jämn som möjligt för ett bättre flyt.

Frametime: Den tid i millisekunder det tar för datorn att rendera en enskild bildruta.

FreeSync: AMD:s teknik för variabel uppdateringsfrekvens som synkroniserar skärmen med grafikkortet.

Fstab: En konfigurationsfil i Linux som beskriver hur och var olika lagringsvolymer ska monteras.

Fullständig behörighet: En NTFS-rättighet som ger en användare total kontroll över en fil eller mapp.

FXAA: En snabb och resurssnål metod för kantutjämning som appliceras efter att bilden är renderad.

GDDR6: En modern typ av höghastighetsminne som används uteslutande på grafikkort.

GC (Garbage Collection): En bakgrundsprocess i SSD-enheter som rensar och optimerar block för att bibehålla prestanda.

GPO (Group Policy): Ett system för central styrning av inställningar och rättigheter i en Windows-domän.

GPU: Graphics Processing Unit; processorn som sköter bildberäkningar och tung parallell matematik.

GRUB: Den vanligaste typen av bootloader som används för att starta Linux-system.

GUI: Grafiskt användargränssnitt; ett system med ikoner och fönster istället för bara text.

G-Sync: Nvidias teknik för variabel uppdateringsfrekvens för att motverka bildsprickor (tearing).

H.264/H.265/AV1: Olika standarder (codecs) för att komprimera och koda video effektivt.

Handskakning (Handshake): Processen där två enheter i ett nätverk förhandlar och kommer överens om regler för kommunikationen.

HBM: High Bandwidth Memory; en typ av staplat minne som ger extremt hög bandbredd på små ytor.

HDD: Hard Disk Drive; en mekanisk hårddisk som lagrar data på roterande magnetiska plattor.

HDR: High Dynamic Range; teknik som möjliggör ett bredare spann av ljusstyrka och färger i bild och video.

Heatpipe: Ett värmerör som använder vätska och ånga för att mycket effektivt transportera bort värme från ett chip.

Heliumdisk: En hårddisk fylld med heliumgas för att minska friktion och möjliggöra fler plattor i samma chassi.

HMAC: En metod för att verifiera ett meddelandes äkthet med hjälp av en hashfunktion och en hemlig nyckel.

Honeypot: En avsiktligt sårbar server eller resurs som används som fälla för att studera angripares beteende.

HSM (Hardware Security Module): En fysisk hårdvaruenhet som används för säker hantering och lagring av digitala nycklar.

HTTP/2: En modern version av webbprotokollet som tillåter flera förfrågningar samtidigt över en anslutning.

HTTP/3 (QUIC): Den senaste versionen av HTTP som använder UDP för att drastiskt minska fördröjningar.

IaaS/PaaS/SaaS: Olika molnmodeller där man hyr infrastruktur, plattformar eller färdig mjukvara som tjänst.

I/O-schemaläggare: En del av operativsystemet som bestämmer i vilken ordning läs- och skrivanrop ska utföras.

ICMP: Ett protokoll som används för diagnostik och felmeddelanden i nätverk, exempelvis genom kommandot ping.

IDPS (IDS/IPS): System som upptäcker och vidtar åtgärder mot misstänkt eller skadlig nätverkstrafik.

IEC/ISO-enheter (decimal): Enhetsmått där 1 kB motsvarar exakt 1 000 bytes (till skillnad från binära KiB).

IEEE 802.1Q: Den internationella standarden för att märka (tagga) trafik som tillhör olika VLAN.

IETF: Den organisation som utvecklar och fastställer öppna standarder för internetprotokoll.

IGPU: En grafikprocessor som är inbyggd direkt i huvudprocessorn eller på moderkortet.

IOMMU: Hårdvarustöd för att isolera enheter och låta virtuella maskiner få direktåtkomst till dem (passthrough).

IOPS: Input/Output Operations Per Second; ett mått på hur många små, slumpmässiga läsningar eller skrivningar en disk klarar per sekund.

IP-kamera: En digitalkamera som skickar sin videoström direkt över ett IP-nätverk.

IPv4: Den dominerande standarden för IP-adresser, baserad på 32 bitar.

IPv6: Den nya standarden för IP-adresser med 128 bitar, vilket ger ett nästintill oändligt antal adresser.

iSCSI: En metod för att skicka SCSI-lagringskommandon över ett vanligt IP-nätverk.

JIT (Just-In-Time): En teknik där källkod översätts till maskinkod i exakt det ögonblick den ska köras.

Journaling: En funktion i filsystem som loggar alla planerade ändringar för att snabbt kunna återställa systemet efter ett strömavbrott.

Kabelkategori (Cat5e/6/6A): Olika klassificeringar av nätverkskablar som avgör vilken hastighet och sträcka de klarar.

Kallstart (cold boot): Att starta upp en dator som har varit helt avstängd och strömlös.

Kerberos: Ett protokoll för säker autentisering i nätverk baserat på digitala "biljetter".

Kill chain: En modell som beskriver de olika stegen i en cyberattack, från planering till slutförande.

Klockfrekvens: Antalet cykler en processor utför per sekund, mäts oftast i Gigahertz (GHz).

Kompilering: Processen att översätta mänskligt läsbar källkod till maskinkod som datorn kan köra.

Kondensator: En elektronisk komponent som tillfälligt lagrar och jämnar ut elektrisk laddning.

Kryptering (EFS): Inbyggd teknik i Windows filsystem NTFS för att kryptera enskilda filer och mappar.

Kylpasta (Thermal Paste): En värmeledande massa som fyller ut mikroskopiska ojämnheter mellan CPU/GPU och dess kylfläns för bättre kylning.

KVM: Kernel-based Virtual Machine; en populär virtualiseringsteknik som är inbyggd i Linux-kärnan.

L1/L2/L3-cache: Olika nivåer av snabbminne i processorn, där L1 är snabbast men minst.

LACP: Ett protokoll som används för att slå ihop flera fysiska nätverksportar till en logisk länk för högre fart.

LAN: Local Area Network; ett lokalt nätverk inom ett begränsat område som ett hem eller ett kontor.

Latency (latens): Den fördröjning eller väntetid som uppstår vid dataöverföring eller beräkning.

LGA1700: Intels processorsockel för 12:e, 13:e och 14:e generationens Core-processorer.

LHR (Lite Hash Rate): En begränsning som Nvidia införde på vissa grafikkort för att göra dem mindre attraktiva för kryptomining.

LLM (Large Language Model): En typ av AI-modell tränad på enorma mängder text för att förstå, generera och översätta mänskligt språk.

LUKS: Den vanligaste standarden för att kryptera hela hårddiskar i Linux-miljöer.

LUN: Logical Unit Number; en identifierare för en logisk lagringsvolym i ett SAN.

LVM: Logical Volume Manager; ett system i Linux för att hantera lagringsvolymer mycket flexibelt.

M.2: En liten kontakt och formfaktor som främst används för snabba SSD-enheter men även för Wi-Fi-kort.

MAC-adress: En unik och permanent fysisk adress som varje nätverkskort har från tillverkningen.

MFA: Multifaktorautentisering; ett system som kräver minst två oberoende metoder för att bevisa någons identitet.

Microcode: Programvara på extremt låg nivå som kan uppdatera eller rätta till felaktigt beteende i en processor.

Micro-USB: En äldre och mindre typ av USB-kontakt som tidigare var standard på många mobila enheter.

Mini-ITX: En mycket kompakt formfaktor för små moderkort och chassin.

MITM (Man-in-the-Middle): En typ av attack där en angripare i hemlighet avlyssnar eller manipulerar trafiken mellan två parter.

MITRE ATT&CK: En omfattande kunskapsbas över de tekniker och taktiker som faktiska angripare använder vid cyberattacker.

MMU: Memory Management Unit; den del av CPU:n som sköter översättningen mellan virtuella och fysiska minnesadresser.

Montera (mount): Processen att koppla samman en lagringsvolym med en specifik mapp i filsystemet så att den blir åtkomlig.

MOSFET: En typ av transistor som ofta används i moderkortets spänningsreglering (VRM) för att styra strömmen till CPU:n.

MSAA: En metod för kantutjämning som tar flera prover per pixel för att skapa mjuka linjer i 3D-grafik.

MTU: Maximum Transmission Unit; den största storleken på ett nätverkspaket som får skickas över en förbindelse.

NAS: Network Attached Storage; en nätverksansluten lagringsenhet som erbjuder fildelningstjänster (t.ex. SMB eller NFS).

NAT: Network Address Translation; en teknik i routrar som låter flera enheter dela på en enda publik IP-adress.

NCQ: Native Command Queuing; en funktion i SATA-diskar som optimerar ordningen på inkommande läs- och skrivanrop.

NFC: Near Field Communication; trådlös teknik för extremt korta avstånd, används för betalningar och parning.

NFS: Network File System; det vanligaste protokollet för fildelning i Unix- och Linux-miljöer.

NUMA: Non-Uniform Memory Access; en teknik i servrar där varje processor har direkt tillgång till sitt eget lokala minne för högre fart.

NVENC/NVDEC: Dedikerad hårdvara på Nvidias grafikkort för att koda respektive avkoda video extremt snabbt.

NVMe: Non-Volatile Memory express; ett modernt protokoll för SSD-enheter som använder PCIe-bussen för extremt låg latens.

NVRAM: Non-Volatile RAM; ett minne som behåller information utan ström, används i UEFI för att lagra inställningar.

OP (Over-provisioning): Att reservera en del av en SSD:s utrymme för att förbättra livslängd och prestanda.

OpenCL/CUDA: Olika API:er som låter grafikkortet utföra vanliga beräkningar som normalt görs av CPU:n.

OSI-modell: En teoretisk modell i sju lager som beskriver hur nätverkskommunikation fungerar, från fysisk kabel till applikation.

Overlays: En teknik där flera lager av filsystem läggs ovanpå varandra, vanligt vid användning av containrar (t.ex. Docker).

Over-voltage: Att medvetet höja spänningen till en komponent för att nå högre klockfrekvenser, vilket ökar både värme och risker.

Over-writing: Att skriva ny data över gammal data på ett lagringsmedium.

Over-clocking: Att öka en komponents klockfrekvens över tillverkarens specifikationer för att få ut mer prestanda.

Packet sniffing: Att använda mjukvara för att fånga upp och i detalj analysera trafiken som skickas i ett nätverk.

Paritet: Extra kontrollinformation som skapas vid lagring (t.ex. i RAID) för att kunna återställa data om en disk går sönder.

Passthrough: Teknik för att låta en virtuell maskin få exklusiv och direkt tillgång till en fysisk hårdvarukomponent.

PBKDF2/Argon2: Algoritmer för att "härda" lösenordshashar genom att göra dem matematiskt tunga att knäcka via gissningar.

PCI: En äldre standard för expansionsplatser i datorer, numera ersatt av PCIe.

PCIe: Peripheral Component Interconnect express; den moderna standarden för snabba instickskort som grafikkort och SSD.

PCIe-banor (lanes): De individuella dataspåren i en PCIe-anslutning som avgör den totala bandbredden (t.ex. x4 eller x16).

PFS (Perfect Forward Secrecy): En säkerhetsfunktion som gör att gamla krypterade sessioner förblir säkra även om den privata nyckeln läcks senare.

Phishing (Nätfiske): Social manipulation där angripare utger sig för att vara en betrodd part för att stjäla känslig information.

PJT (Project): En organiserad arbetsyta eller mappstruktur för utvecklingsprojekt.

PKI: Public Key Infrastructure; ett system med digitala certifikat och betrodda utfärdare för säker identifiering på internet.

PoE: Power over Ethernet; teknik för att skicka både ström och data genom en enda nätverkskabel.

Port forwarding: En inställning i en router som skickar trafik från en specifik extern port till en viss enhet i det interna nätverket.

POST: Power-On Self-Test; det hårdvarutest som datorn utför omedelbart vid uppstart för att kontrollera att allt fungerar.

Power budget: Den maximala mängd elektrisk effekt som finns tillgänglig för en specifik komponent eller portgrupp.

PowerShell: Ett avancerat skriptspråk och kommandotolk i Windows för professionell systemadministration.

PPT (AMD-gräns): Package Power Tracking; den absoluta effektgränsen i watt för AMD:s Ryzen-processorer.

PSU: Power Supply Unit; nätaggregatet som omvandlar växelström från vägguttaget till likström för datorns delar.

PXE-boot: Teknik för att starta upp och installera operativsystem på en dator direkt över nätverket utan lokal disk.

QCOW2: Ett flexibelt filformat för virtuella hårddiskar som stödjer komprimering och snapshots.

QOS: Quality of Service; tekniker som används för att prioritera viktig trafik (som röstsamtal) framför mindre tidskritisk trafik.

QLC: Quad-Level Cell; en typ av flashminne som lagrar fyra bitar per cell, vilket ger hög kapacitet till lågt pris.

Qubit: Den minsta informationsbäraren i en kvantdator som kan existera i flera tillstånd samtidigt genom superposition.

RAID: Redundant Array of Independent Disks; att kombinera flera hårddiskar för att uppnå högre fart eller säkerhet.

RAID 0: Striping; data sprids över diskar för hög fart, men om en disk går sönder försvinner all data.

RAID 1: Spegling; all data skrivs identiskt till två diskar för hög säkerhet.

RAID 5: Data och kontrollinformation sprids över minst tre diskar; systemet tål att en valfri disk går sönder.

RAID 6: Liknar RAID 5 men med dubbel kontrollinformation; systemet tål att två diskar går sönder samtidigt.

RAID 10: En kombination av spegling och striping för att få både hög prestanda och hög säkerhet.

RAM: Random Access Memory; datorns snabba arbetsminne där data lagras tillfälligt medan datorn är igång.

Ransomware: Skadlig kod som låser eller krypterar offrets data och kräver en lösensumma för att låsa upp den.

RBAC: Role-Based Access Control; ett system där användares åtkomst styrs av vilken roll de har i organisationen.

Rec.709/Rec.2020: Standarder för hur färger ska representeras i HD- respektive UHD-video.

Recovery Environment: En speciell startmiljö som används för att reparera eller återställa ett skadat operativsystem.

ReFS: Resilient File System; Microsofts moderna filsystem designat för hög dataintegritet och stora datamängder.

Regedit: Det grafiska verktyget som används för att titta i och ändra i Windows-registret.

Registry hive: En huvudavdelning i Windows-registret som lagras som en egen fil på hårddisken.

Reset (CMOS-clear): Att återställa moderkortets inställningar till fabriksstandard genom att nollställa CMOS-minnet.

Resursövervakaren: Ett verktyg i Windows som visar detaljerad information om CPU, minne, disk och nätverk i realtid.

RJ45: Den standardiserade kontakttypen som används för vanliga nätverkskablar av koppar.
